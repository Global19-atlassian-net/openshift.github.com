Overview

Describes an Cartridge Example:

```
<cartridge>
  <name>mysql-5.1</name>
  <type>embedded</type>
  <status-messages nil="true"/>
  <properties>
    <connection-url>mysql://127.0.250.129:3306/</connection-url>
    <username>foo</username>
    <password>bar</password>
    <database-name>baz</database-name>
  </properties>
  <links>
  ...
  </links>
</cartridge>
```
