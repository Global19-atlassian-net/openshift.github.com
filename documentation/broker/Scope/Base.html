- (Object) limits_access(criteria)

Given a Mongoid query criteria, apply a set of restrictions to the query object. Since many scopes may be queried, set the following options on the criteria to allow post processing

:for_ids - a list of IDs that are allowed to be viewed by this
           scope.  If set, no other IDs are returned.
:visible - set to true if any results should be visible, or
           ||= false if the scope does not grant access

The caller is responsible for limiting the Mongoid criteria appropriately for the user's access.
